# Expense Tracker

Автор: катя сергеева

## Описание

Это графическое приложение для отслеживания личных расходов. Вы можете добавлять расходы с указанием суммы, категории и даты. Также есть возможность фильтрации по категории и диапазону дат, а также подсчёта общей суммы за выбранный период.

## Использование

1. Запустите программу.
2. Введите сумму, выберите категорию и дату, нажмите «Добавить расход».
3. Используйте фильтры для просмотра расходов по категориям и датам.
4. Общая сумма отображается автоматически.

## Примеры тестов

- Добавление расхода 500 рублей на еду 2023-09-20.
- Фильтрация по категории «Транспорт».
- Фильтрация по диапазону дат: с 2023-09-01 по 2023-09-30.
